题目描述:
输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

题目解析:
使用了两种方法，第一种方法是自己在OJ时使用的循环，处理完公共部分后，再把剩下的一个链表的部分添加到最后合并链表的末尾。    
后来看了《剑指offer》，上面采用的递归的方式，代码更简洁一些。
