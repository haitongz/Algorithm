题目描述:
请实现一个函数，检查一棵二叉树是否为二叉查找树。
给定树的根结点指针TreeNode* root，请返回一个bool，代表该树是否为二叉查找树。

题目解析:
根据二叉查找树的中序遍历结果是升序的特点判断。
