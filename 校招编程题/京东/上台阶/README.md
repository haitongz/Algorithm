题目描述:
有一楼梯共m级，刚开始时你在第一级，若每次只能跨上一级或者二级，要走上m级，共有多少走法？注：规定从一级到一级有0种走法。
给定一个正整数int n，请返回一个数，代表上楼的方式数。保证n小于等于100。为了防止溢出，请返回结果Mod 1000000007的值。

测试样例:
3

返回: 2

题目解析:
一道简单的动态规划题，因为每次只能跨上一级或二级，所以动态规划方程为: s[i]=s[i-1]+s[i-2]；剩下的就很简单了。
