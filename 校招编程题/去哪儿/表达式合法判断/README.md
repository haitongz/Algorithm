题目描述:
写一段代码，判断一个包括'{','[','(',')',']','}'的表达式是否合法(注意看样例的合法规则。)
给定一个表达式A,请返回一个bool值，代表它是否合法。

测试样例:
"[a+b*(5-4)]*{x+b+b*(({1+2)}}"

返回: false

题目解析:
相信所有学过栈这种数据结构的同学都记得，表达式判断是栈的一个重要应用，这道题更简单一题，只需考虑题中给出的几种符号是否满足匹配条件即可。
