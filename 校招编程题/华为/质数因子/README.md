题目描述:
功能:输入一个正整数，按照从小到大的顺序输出它的所有质数的因子（如180的质数因子为2 2 3 3 5 ）
最后一个数后面也要有空格

输入描述：
输入一个long型整数

输出描述：
按照从小到大的顺序输出它的所有质数的因子，以空格隔开。最后一个数后面也要有空格。

输入例子：
180

输出例子：
2 2 3 3 5

题目解析:
代码比较简单，主要是质数因子的求法，数学问题。
