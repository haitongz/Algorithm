题目描述:
找出字符串中第一个只出现一次的字符

输入描述：
输入一个非空字符串

输出描述：
输出第一个只出现一次的字符，如果不存在输出-1

输入例子：
asdfasdfo

输出例子：
o

题目解析:
使用数组来存储每个字符出现的次数，数组下标为每个字符与'a'相减的差值。
