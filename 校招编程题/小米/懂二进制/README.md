题目描述:
世界上有10种人，一种懂二进制，一种不懂。那么你知道两个int32整数m和n的二进制表达，有多少个位(bit)不同么？ 

测试样例:
1999 2299

返回: 7

题目解析:
考点有二，一是异或运算（结果为1表示位上的数不同，结果为0表示相同），二是对结果数1的个数进行统计。
